# Obervateurs

Bien que toutes les ressources soient accessibles sur la plateforme, il est nécessaire de pouvoir choisir lesquelles suivre afin de rester informé des dernières modifications
et de pouvoir les retrouver plus facilement. Pour cela, PLaTon propose un système d'observateurs.

## Devenir observateur

- Pour devenir observateur d'une ressource, il suffit de cliquer sur le bouton "Suivre" présent sur la page de la ressource.
- Une fois observateur, il est possible de se désabonner en cliquant sur le bouton "Ne plus suivre" présent sur la page de la ressource.
- Lorsqu'on créé une ressource, on devient automatiquement observateur de celle-ci.

## Avantages

- Pour un exercice, suivre l'exercice permet de l'ajouter à des activités.
- Pour une activité, suivre l'activité permet de l'ajouter à des cours.
- Des notifications sont envoyées en cas de changement important (changement de status, ajout d'activités et exercices, etc.)

<Callout type="warning">
  Seules les ressources suivies sont ajoutables dans les cours et activités pour des raisons de performances.
</Callout>
